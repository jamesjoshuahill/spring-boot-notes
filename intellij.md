# IntelliJ

## Keyboard shortcuts

### Navigation
- `⌘O` : open class
- `⇧⌘O` : open file
- `⌘E` : open recent file
- `⌘↑` : jump to navbar

### Jump to
- `⌘B` : go to declaration

### Refactor
- `⌘T` : refactor this
- `⇧F6` : rename...

### Code
- `⌥⌘L` : reformat file
- `⇧⌘Enter` : complete method braces (from inside method parameter list)
- `⌘D` : duplicate line
- `⇧⌥↑` : move line up
- `⇧⌥↓` : move line down
- `⌥⌘↑` : move statement up
- `⌥⌘↓` : move statement down

## Shorthands

- `psvm` : `public static void main()`...
